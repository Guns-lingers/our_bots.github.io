Сосистер Бит

Бот для воспроизведения музыки с платформы YouTube

Описание

Бот может воспроизводить музыка по ссылке с платформы YouTube

Может осуществлять поиск, а так же воспроизводить свои сохраненные плей-листы

Есть возможность создавать свои плей-листы,
которые будут работать на всех серверах где есть данный бот

Что нового добавлено

Добавлена возможность скачивание композиций с платформы YouTube
